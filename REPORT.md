# parallel-programming
# Отчёт по лабораторной работе: Перемножение матриц на C++ с верификацией на Python

## 1. Задание
Реализовать программу для перемножения двух квадратных матриц на языке C++,
а также автоматизированную верификацию результатов вычислений с помощью Python.

## 2. Описание работы скриптов

### 2.1. `generate_matrices.py`
- Использует `numpy.random.randint` для генерации целочисленных значений.
- Порядок матрицы `n_matrix` задан внутри скрипта (`n_matrix = 1000`).
- Записывает `n_matrix` в первую строку текстового файла, затем сгенерированный элементы матрицы построчно.

### 2.2. `matrix_multiplication.cpp`
- Матрицы хранятся в `vector<long long>` как одномерный массив.
- Доступ к элементу `(i, j)` осуществляется по формуле `i * N + j`, где `N` - это порядок квадратной матрицы.
- Для измерения времени выполнения задачи используется `chrono::high_resolution_clock`.
- Результат и метаданные (объём задачи (порядок квадратной матрицы), время) записываются в `result_C.txt`.

### 2.3. `verification_matrix.py`
- Загружает матрицы A, B и результат C++ из файлов по фиксированным путям.
- Считывает порядок квадратных матриц `n_matrix` из первой строки каждого файла.
- Вычисляет эталонное произведение через `np.dot` и выполняет точное поэлементное сравнение с результатом C++ с помощью `np.array_equal`.
- При успешном совпадении сохраняет эталонную матрицу в файл `verification_result_C.txt` для возможности визуального анализа и выводит подтверждение в консоль.
- В случае ошибки выводит уведомление и завершает работу.

## 3. Результаты экспериментов

Для демонстрации работы программы были проведены эксперименты с матрицами порядка 100, 600, 1000.

### 3.1. При порядке равном 100
Время выполнения задачи: 5.3889ms

### 3.2. При порядке равном 600
Время выполнения задачи: 1022.38ms

### 3.3. При порядке равном 1000
Время выполнения задачи: 4801.54ms


## 4. Выводы
- Программа успешно перемножила матрицы и записала результат вместе с метаданными (размер задачи и время выполнения) в файл result_C.txt.
- Python скрипт верификации подтвердил, что результаты, полученные C++ программой, полностью совпадают с эталонными вычислениями, выполненными с помощью библиотеки NumPy.
- Для квадратных матриц, порядок которых не превосходит 100, время выполнения C++ программы составило порядка меньше 0.0053889 секунд, что демонстрирует высокую скорость на малых размерах.
- Для квадратных матриц, порядок которых не превосходит 600, время выполнения C++ программы составило в переделах одной секунды.
- Для квадратных матриц, порядок которых свыше 600, время выполнения C++ программы время выполнения значительно увеличивается из-за кубической зависимости сложности алгоритма O(N^3).
